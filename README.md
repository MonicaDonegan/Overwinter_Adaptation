# Overwinter_Adaptation
Code for Analysis for 2020-2023 HREC Overwinter Project
- Data folders
  1. "Data_files" includes experimental setup metadata and symptom, stunting, temperature, culturing data. 
  2. "qpcr_files" includes all raw qPCR data from the field project.
- R code 
  1. "qpcr_processing_adaptation.R" - functions to load and process qPCR raw files
  2. "input_datasets.R" - code to input, clean, and process data types for analysis and visualization
  3. "graphing_adaptation.R" - code for visualization for all main and supplementary figures
  4. "models_adaptation.R" - code for all statistical models in the manuscript
